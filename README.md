# MovieClub

Welcome to MovieClub, a project for managing and organizing movie-watching activities for a group. This repository contains the source code and documentation for the application.

## About The Project

MovieClub is designed to streamline the experience of running a movie club. It helps members suggest films, vote on what to watch next, and keep a shared history of movies watched together.

This project is currently in the initial stages of development. Further instructions on setup and usage will be added as the project matures.
